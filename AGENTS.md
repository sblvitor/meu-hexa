# Meu Hexa

MeuHexa is an interactive web platform about the 2026 FIFA World Cup, aimed at Brazilian fans. The system has three main features:

1. **Squad Selection** — The user builds their Brazilian national team by selecting 26 players from a pre-registered list, with filters by position and search by name. Upon completion, a stylized card is generated for sharing.

2. **Simulator** — The user simulates the World Cup results: fills in scores for the group stage (with automatic qualification calculation) and the knockout rounds (round of 16 through the final, with penalty shootout support). At the end, a visual summary of the campaign is generated for sharing.

3. **Tier List** — The user organizes the 48 World Cup teams into customizable tiers (S, A, B, C, D, F) via drag-and-drop. Upon completion, a card with the complete tier list is generated for sharing.
